Etquetas de Texto

La etiqueta "p" define un párrafo de texto.

Fragmentos de texto

La etiqueta "strong" define un ragmento de texto importante o palabras clave.

La etiqueta "em" define un fragmento de texto enfatizado respecto a la frase que lo contiene.

La etiqueta "mark" define un Fragmento de texto resaltado

La etiqueta "i" define texto en cursiva.

La etiqueta "b" define texto en negritas.

La etiqueta "u" define texto subrayado.

La etiqueta "s" define texto cancelado.

La etiqueta "span" define texto seleccionado.

La etiqueta "cite" define citado de algun texto.

Modificación de significado

La etiqueta "sup" define el superindice , por ejemplo 42.

La etiqueta "sub" define el subindice , por ejemplo 42.

La etiqueta "small" define un texto de menor tamaño.

La etiqueta "q" Cita o frase extraída de otro contexto, requiere del atributo cite.
